# active voice

active voice is the default. passive voice is used deliberately, not by accident.

## the rule

write "we built this" not "this was built by us". write "users complete the form" not "the form is completed by users".

## when passive voice is right

- when the agent is unknown or unimportant: "the file was deleted" (we don't need to say who)
- when the agent is obvious and emphasizing it would distract: "you'll be notified when it's ready" (the agent is "we", but the user doesn't need to think about that)
- when the recipient is the subject of the larger paragraph: "the report was reviewed by three teams. it was approved on tuesday." (the report is the subject; the agents are secondary)

## when passive voice is wrong

- when you're avoiding ownership: "mistakes were made" (own the mistake)
- when it makes the sentence longer for no reason
- when it removes the agent entirely and the agent matters

## red flags

- "is being [verb-ed]"
- "has been [verb-ed]"
- "will be [verb-ed]"
- a lot of "by [agent]" constructions

if a paragraph has three or more passive constructions, the paragraph is dragging. rewrite at least one in active voice.

## what to avoid

- using passive voice to sound more professional (it doesn't)
- using passive voice to hide who did something (readers notice)
- mixing active and passive in the same sentence ("the decision was made and we'll communicate it")
