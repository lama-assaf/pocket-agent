# decisions

one file per significant project decision. lightweight ADR format.

filename: `YYYY-MM-DD-short-slug.md`

template:

```
# decision: [short title]

date: YYYY-MM-DD
status: [proposed | accepted | superseded by [other decision]]

## context

what's happening that requires a decision.

## decision

what we decided.

## alternatives considered

what else we looked at and why we didn't pick it.

## consequences

what we expect to gain. what we explicitly give up. what we'll watch for.

## review

when we'll re-evaluate this. what would trigger a revisit.
```

---

decisions are easier to remember than discussions. write them down. revisit them when context changes.
