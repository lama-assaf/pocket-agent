# instincts

short, durable patterns to apply across every atelier session.

this file is a template. add your own. remove what doesn't apply. keep entries short.

---

## examples

(delete these once you have your own.)

- **stack defaults**: when generating code, prefer typescript + node 22 with `--experimental-strip-types`. no transpilers when avoidable.
- **package naming**: scoped names by default. unscoped only when verified available on npm.
- **commit voice**: lowercase, imperative, no exclamation marks.
- **review cadence**: design system audit runs at every major version bump.
- **never**: never write a prd without naming the evidence. if there is none, write "no evidence, validate before build" — don't skip the line.
- **handoff**: handoff docs include states, edges, and a screenshot. one of those missing = handoff isn't done.
