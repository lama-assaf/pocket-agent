# /tagline

write taglines, slogans, or hero headlines.

## what it does

generates 15-25 options across four jobs:

- describe (says what it is)
- position (defines what it is against alternatives)
- promise (signals an outcome)
- provoke (gets attention with the unexpected)

narrows to 5-7 strong candidates. each with rationale.

## how to use

```
/tagline
```

then provide: product, audience, feeling you want, what to position against.

## rules it follows

- short (3-8 words for most, up to 12 if rhythm demands)
- specific (a stranger should be able to guess what you do)
- bans common patterns ("the future of", "X simplified", "X reimagined")
- read aloud test built in

## output

candidates + top 5 with rationale + final recommendation.
