# /landing

write copy for a landing page. invokes landing-copy skill.

## what it does

builds:

- hero (headline, subhead, primary cta, secondary cta, visual direction)
- problem section
- solution section
- 2-4 feature sections (each: claim + body + evidence)
- social proof block
- objection handling
- final cta
- footer

each section gets 2-3 headline options.

## how to use

```
/landing
```

then provide:

- product
- audience
- voice guide (or examples)
- proof you can use (numbers, customers, integrations)
- the primary action you want visitors to take

## output

structured copy by section + a single rendered draft to start from.

## what it avoids

- "the future of"
- "join thousands" (without a number)
- adverbs doing the work of evidence
- testimonials without names or titles
- carousels
